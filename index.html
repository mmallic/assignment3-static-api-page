Group 5 API Documentation

User Management Service

User Registration POST /api/auth/register

Register a new user

User Authentication POST /api/auth/login

Authentes a user and returns a JWT token.

Resource Catalog Service

Retrieve Extension List GET /api/extensions

Returns a list of available extensions with filtering options.

Retrieve Extension details GET /api/extensions/{id}

Returns details of a specific extension.

Add New Extension POST /api/extensions

Allows Authenticated users to add a new extension.

Update Extension information PUT /api/extensions/{id}

Updates extension details, including archiving.

Delete an Extension. DELETE /api/extensions/{id}

Removes an extension from the catalog.

File upload Service

Upload an Extension File. POST /api/file/upload

Uploada an extension file to the server

Download an Extension File. GET /api/file/download/{id}

Downloads an extension file and tracks download count.
